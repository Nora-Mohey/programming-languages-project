A program is a set of instructions according to Rules of the language of choice. Computers are useless without programs. A software is sort of like a recipe. It contains a list of ingredients (called variables) and a list of directions (called statements) that tell What to do with the variables by computer. Programing language : A formal language Designed to write Computer Instructions Carried out specific tasks. In general, the term programming language refers to For languages of high standard, such as BASIC, C, C++, Java, Pascal and Fortran .In addition to a specific method of writing (Syntax), it specifies how program commands are organized and source code .
